Privacy Policy

This app helps users generate marketplace listing content from uploaded product photos.

Information Used

  • Product images uploaded by the user
  • Listing details entered by the user

How Information Is Used

Information is used only to generate listing content, pricing suggestions, and optional marketplace draft actions requested by the user.

Storage

No long-term user marketplace account data is intentionally stored by this static site. Local app behavior depends on the user’s own environment and configuration.
